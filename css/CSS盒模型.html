CSS盒模型包括：边距、边框、填充、实际内容。

  • margin(外边距)：清除边框外的区域，外边距是透明的
  • Border(边框):围绕在内边距和内容外的边框
  • Padding(内边距):清除内容周围的区域，也是透明的
  • Content(内容):盒子内容

CSS的两种盒模型 W3C 和 IE

  • 在标准盒模型系中，盒模型的宽度只是内容的宽高
  • 在IE 中盒模型的宽高是内容（content）+ 填充(Padding) + 边框(Border)的总宽高

标准模型： box-sizing:content-box; width = content

IE模型： box-sizing:border-box； width = border + padding + content
